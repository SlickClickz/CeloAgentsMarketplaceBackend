The Celo Agent Marketplace (CAM) is a permissionless, open infrastructure layer that enables the discovery, verification and economic coordination of autonomous agents on Celo. CAM is not a product, it is a public good that any wallet, dApp, orchestrator or agent can integrate with through a simple REST API.
CAM composes four existing Celo primitives into a unified stack: 
ERC-8004 (identity), Celo/skills (capabilities)
AgentScan (execution history)
x402 via Thirdweb (payments). 
No new standards are introduced. CAM's value is in how it connects them.
